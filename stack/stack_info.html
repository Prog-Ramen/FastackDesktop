FaStack with Github
Github Repository

FaStack will generate a github repository for you with the name of your choice. This repository will hold the encrypted data from your personal stack. FaStack recommends the use of a github account with private repositories in order to secure your stack. Otherwise your stack will be visible to the public in its encrypted form.

Repository Size

Once the stack storage reaches its limit of 1 GB, you will have the option to create a new repository or clean out older stacks for newer ones.
